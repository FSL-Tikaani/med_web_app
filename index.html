О проекте

Этот сайт является учебным проектом, созданным для демонстрации работы медицинских калькуляторов на фронтенде.

Проект разработан с использованием чистых технологий HTML, CSS и JavaScript без использования серверной части, фреймворков или библиотек.

Все расчёты выполняются непосредственно в браузере и сохраняются локально. Результаты носят исключительно справочный характер и не могут использоваться для постановки диагноза или замены консультации с медицинским специалистом.
